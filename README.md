# 🏛️ House of Quants

**One team. One budget. Multiple revenue streams.**

A collaborative AI trading team where specialized agents share a capital pool and coordinate across crypto trading, prediction markets, and research — with token costs as real capital drag.

- **Frontend:** Vite + React + Tailwind (dark theme, emerald accents)
- **Backend:** Convex (real-time DB, shared with Mission Control)
- **LLM:** OpenRouter (multi-model)
- **Data:** MCP tool servers + API Bridge
- **Deploy:** Vercel (auto-deploy on push)
